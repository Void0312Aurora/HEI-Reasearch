# HEI Curriculum Learning

This directory contains the implementation of the staged curriculum learning for SoulEntity.

## Structure

- `stage0_existence.py`: Level 0 - Existence Training (Dynamics Stability)
- `stage1_counting.py`: Level 1 - Counting & Periodicity
- `stage2_fsm.py`: Level 2 - Finite State Machine
- `common/`: Shared utilities for curriculum training

## Usage

Run the scripts directly from the root directory:

```bash
python HEI/curriculum/stage0_existence.py
```
